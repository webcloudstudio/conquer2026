Conquer v5

A modern revival of the classic 1987 multi-player turn-based strategy game.

What is Conquer? Conquer is a game of empire building, diplomacy, and conquest played on a procedurally generated hex map. Each player controls a nation, managing resources, armies, and magic across a series of turns to achieve their unique victory condition.

Key Features

  • Hex map strategy — Explore and claim procedurally generated terrain
  • 5 resource types — Talons (gold), jewels, metals, food, and wood
  • 8 asymmetric player classes — Each with unique units and victory conditions
  • 3 magic power types — Military, civilian, and wizardry
  • Turn-based multiplayer — Submit orders between turns; game advances on a schedule
  • NPC nations — Computer-controlled factions fill the map from the start
  • World events — Droughts, earthquakes, plagues, and magical surges
  • Artifacts — Rare items scattered across the map with powerful effects
  • Diplomacy — Form alliances, declare war, or send tribute

Quick Start

  1. See Running the Game to get the server running
  2. Register an account at the login screen
  3. Pick an open world from the game browser
  4. Choose your nation name, race, and player class
  5. Read Gameplay Guide to understand your first turn

Origin

Conquer was written in C in 1987 by Ed Barlow and Adam Bryant for USENET distribution, and ran as a play-by-email game on Unix systems for years. This version is a GPL-licensed modern rewrite in Python + React, preserving the original mechanics while completing systems that were never finished in the original (notably the NPC AI).
